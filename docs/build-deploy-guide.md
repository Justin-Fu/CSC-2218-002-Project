1. Project Setup

Due to the simplicity of this project, there is not many software requirements required to a person needs to run this project. On the development side, the developer would need a working computer device with internet connection, a web browser, and a code editor, such as Microsoft Visual Studio Code. A user would need a working computer device with internet connection, and a web browser, in order to access the customer section of this project, which is just a webpage, so there is not much additional barriers that may prevent easy access for users.


2. Build Process

The system would be build and hosted on the platform GitHub, and all change history and development history can be easily tracked using the platform.


3. Deployment Process

The application would be deployed to the hosted repository on GitHub, and if this were to be a real project, then it would be deployed to a website, controlled by the project owner, such a web server.
